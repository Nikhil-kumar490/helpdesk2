Data science and Big data

Supervised vs. Unsupervised Learning

Machine learning is broadly classified into two types: Supervised Learning and Unsupervised Learning.

In Supervised Learning, the model learns from labeled data, meaning each input has a corresponding correct output. The goal is to learn a mapping function that can predict the output for new inputs. It is mainly used for tasks like regression (predicting continuous values, such as house prices) and classification (predicting discrete categories, such as spam or not spam). Common algorithms include Linear Regression, Decision Trees, Support Vector Machines (SVM), and Neural Networks.

On the other hand, Unsupervised Learning works with unlabeled data, meaning the model tries to identify patterns and structures in the data without predefined answers. It is used for tasks like clustering (grouping similar data points, such as customer segmentation) and association rule learning (finding relationships between data, like market basket analysis). Common algorithms include K-Means Clustering, Hierarchical Clustering, and Principal Component Analysis (PCA).

Regression vs. Classification

Both regression and classification are types of supervised learning, but they have different objectives.

Regression is used when the output variable is continuous. It aims to predict a numeric value based on input features. For example, predicting house prices based on square footage and location is a regression task. The most common regression algorithm is Linear Regression, where the relationship between input and output is modeled as a straight line:

Y = mX + c

Classification, on the other hand, is used when the output variable is categorical. The goal is to assign the input data to one of several predefined classes. For example, an email spam filter classifies emails as spam or not spam. Common classification algorithms include Decision Trees, Logistic Regression, and Random Forest.

Numerical Concepts

Mean (Average)

The mean is the sum of all data points divided by the number of data points. It is a measure of central tendency, showing the "average" value of a dataset:

Mean = (ΣX) / n

Variance

Variance measures how spread out the data points are around the mean. A high variance means the data points are widely spread out, while a low variance means they are closely clustered around the mean:

Variance = (Σ (X - Mean)²) / n

K-Nearest Neighbors (KNN)

K-Nearest Neighbors (KNN) is a simple yet powerful machine learning algorithm used for both classification and regression. It works by finding the K nearest data points to a given input and making predictions based on the majority class (for classification) or the average value (for regression).

Regression (Basic Concept)

Regression is used to predict continuous values based on input variables. The simplest form is Linear Regression, which assumes a linear relationship between the input and output:

Y = mX + c

Data Science Basics

Types of Data

Data can be classified into three main types:

  • Structured Data – Organized and stored in tables, databases, or spreadsheets (e.g., Excel, SQL databases).
  • Unstructured Data – Data without a predefined format, including text, images, videos, and audio files.
  • Semi-Structured Data – A mix of structured and unstructured data, such as JSON and XML files.

Exploratory Data Analysis (EDA)

Exploratory Data Analysis (EDA) is the process of analyzing and visualizing data to understand its key characteristics before applying machine learning models. It involves several key steps:

  • Understanding Data – Looking at summary statistics such as mean, median, variance, and standard deviation.
  • Visualizing Data – Using histograms, scatter plots, and box plots to understand distributions and relationships.
  • Handling Missing Values – Filling missing values with mean, median, or mode or removing rows with missing data.
  • Detecting Outliers – Identifying outliers using box plots or standard deviation methods.

EDA is a crucial step before training machine learning models, as it helps in making data-driven decisions and improving model accuracy.
